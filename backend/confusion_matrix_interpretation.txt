The confusion matrix is a 10x10 grid where:

Rows = the actual digit
Columns = what the model predicted
Diagonal = correct predictions (you want these dark)
Off-diagonal = mistakes (you want these empty/white)

So if row 7, column 1 is dark, it means the model keeps misreading 7s as 1s -- and now know to draw more distinctive 7s in next data collection session.
